Smiley
======

###### How to compile

```
g++ -o smiley smiley.cpp -lX11 -lGL -lGLU -lGLEW -lSOIL
```

###### Keyboard Shortcuts
- Press ```Esc``` key to quit.
- Press ```F``` or ```f``` to toggle full-screen mode.
