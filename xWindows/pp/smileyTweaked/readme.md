Smiley Tweaked
==============

###### How to compile

```
g++ -o smileyTweaked smileyTweaked.cpp -lX11 -lGL -lGLU -lGLEW -lSOIL
```

###### Keyboard Shortcuts
- Press ```Esc``` key to quit.
- Press ```F``` or ```f``` to toggle full-screen mode.
