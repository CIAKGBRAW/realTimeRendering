Triangle by Orthographic Projection
===================================

###### How to compile

```
g++ -o triangleOrtho triangleOrtho.cpp -lX11 -lGL -lGLU -lGLEW
```

###### Keyboard Shortcuts
- Press ```Esc``` key to quit.
- Press ```F``` or ```f``` to toggle full-screen mode.
