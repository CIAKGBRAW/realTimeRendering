Triangle by Perspective Projection
==================================

###### How to compile

```
g++ -o trianglePerspective trianglePerspective.cpp -lX11 -lGL -lGLU -lGLEW
```

###### Keyboard Shortcuts
- Press ```Esc``` key to quit.
- Press ```F``` or ```f``` to toggle full-screen mode.
