# OpenGl Window

###### How to compile

```
g++ -o extensions extensions.cpp -lX11 -lGL -lGLU -lGLEW
```

###### Keyboard Shortcuts

-   Press `Esc` key to quit.
-   Press `F` or `f` to toggle full-screen mode.
