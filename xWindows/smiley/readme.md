Smiley
======

###### How to compile

- Compile the program.

```
g++ -o smiley smiley.cpp -lX11 -lGL -lGLU -lSOIL

```

###### Keyboard shortcuts
- Press ```Esc``` key to quit.
- Press ```f``` key to toggle fullscreen mode.
