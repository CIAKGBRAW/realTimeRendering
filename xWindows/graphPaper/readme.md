Graph Paper
===========

###### How to compile

- Compile the program.

```
g++ -o graphPaper graphPaper.cpp -lX11 -lGL -lGLU

```

###### Keyboard Shortcuts
- Press ```Esc``` key to quit.
- Press ```F``` or ```f``` to toggle full-screen mode.
