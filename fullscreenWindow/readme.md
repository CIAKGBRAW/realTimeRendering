Fullscreen Window
============

###### How to compile

```
cl.exe /EHsc fullscreenWindow.cpp user32.lib kernel32.lib gdi32.lib
```

###### Keyboard shortcuts
- Press ```Esc``` key to quit.
- Press ```f``` key to toggle fullscreen mode.
