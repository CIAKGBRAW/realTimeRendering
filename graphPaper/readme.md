Graph Paper
======================

###### How to compile

```
cl.exe /EHsc /D:UNICODE /DEBUG graphPaper.cpp user32.lib kernel32.lib gdi32.lib openGL32.lib
```

###### Keyboard shortcuts
- Press ```Esc``` key to quit.
- Press ```f``` key to toggle fullscreen mode.
