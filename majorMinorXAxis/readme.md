Major and Minor X Axis
======================

###### How to compile

```
cl.exe /EHsc /DUNICODE /Zi majorMinorXAxis.cpp user32.lib kernel32.lib gdi32.lib openGL32.lib
```

###### Keyboard shortcuts
- Press ```Esc``` key to quit.
- Press ```f``` key to toggle fullscreen mode.
