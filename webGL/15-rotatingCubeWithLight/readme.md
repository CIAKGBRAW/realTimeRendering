# Rotating Cube with Light

###### Keyboard shortcuts

*   Press `Esc` key to quit.
*   Press `f` key to toggle fullscreen mode.
*   Press `1` to `9` key to increase the speed.
*   Press `l` key to enable lighting.
*   Press `a` key to enable animation.
